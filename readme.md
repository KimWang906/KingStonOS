이 OS는 사이드 프로젝트로 저희가 공부를 하며 개발하는 **ARM기반 임베디드 OS**입니다.  

## init

아래 명령어를 통해 기본 개발환경을 설정하세요(**Ubuntu 22.04.1** 기준)  
다음은 `.sh` 스크립트에 적용되는 명령어입니다.  
  
```sh
chmod +x tools/autoinstall.sh
chmod +x boot/EntryS_Build.sh
```
